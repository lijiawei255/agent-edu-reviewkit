## 快速诊断命令

```bash

grep -c "data:image" 文档.html

# 检查闪卡bug（应无输出）
grep -c "onclick" 文档.html

# 检查公式完整（应为偶数）
grep -o '$$' 文档.html | wc -l

# 检查作者名泄露（应无输出）
grep -i "jiawei\|peng.chen\|haoxuan" 文档.html
```

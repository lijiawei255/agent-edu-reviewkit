# 文科课件提取规则

## 适用范围

本规则适用于以下文科/社科课程类型：
- **语言文学类**：语言学、文学史、写作
- **历史/哲学类**：历史、哲学、思想史
- **社会科学类**：社会学、政治学、经济学、心理学
- **管理/法律类**：管理学、法学、公共政策
- **艺术/设计类**：艺术史、设计理论、美学
- **教育/传媒类**：教育学、传播学、新闻学

当课件内容出现以下特征时，自动切换至文科提取规则：
- 以概念定义和文字论述为主，而非公式推导
- 包含时间线/历史发展脉络
- 包含案例分析和论述题
- 包含大量对比、分类、归纳

## 1. 图片过滤规则

### 1.1 必须丢弃的图片类型

与理科规则相同的基础过滤：
- Logo、页眉页脚、纯装饰图、感谢页
- 分辨率过低无法辨认的图片
- 重复图片（仅保留首次出现）

### 1.2 必须保留的图片类型（文科特化）

| 优先级 | 类型 | 判定特征 |
|--------|------|---------|
| 🔴 P0 | 概念关系图/思维导图 | 展示概念之间的层级、因果、对比关系 |
| 🔴 P0 | 时间线/年代图 | 按时间顺序排列的历史事件或发展阶段 |
| 🔴 P0 | 分类框架图 | 将概念/流派/方法按维度分类的图表 |
| 🟡 P1 | 人物/场景照片 | 展示关键人物、历史场景、艺术作品 |
| 🟡 P1 | 地理/分布图 | 展示空间分布、地域关系 |
| 🟢 P2 | 案例示意图 | 展示具体案例的构成要素 |

## 2. 概念提取规则

### 2.1 概念提取模板

每个重要概念按以下结构记录：

```
📌 概念名称（英文/外文对应词）
定义：[课件中的原话定义]
通俗理解：[用日常类比解释]
核心要素：[拆解为2-5个关键要素]
与其他概念的关系：
  - 前置概念：[需要先理解什么]
  - 并列概念：[和什么容易混淆，区别在哪]
  - 后续概念：[这个概念是理解什么的基础]
现实案例：[这个概念在现实中的体现]
```

### 2.2 多概念辨析

当两个或多个概念容易混淆时，使用对比表格：

```
| 对比维度 | 概念A | 概念B | 概念C |
|----------|-------|-------|-------|
| 定义 | ... | ... | ... |
| 核心特征 | ... | ... | ... |
| 适用场景 | ... | ... | ... |
| 经典代表 | ... | ... | ... |
```

## 3. 时间线/历史脉络处理

### 3.1 识别时间线内容

扫描课件时通过以下特征识别时间线：
- 连续的年份/日期标记
- "第一阶段...第二阶段..."等阶段划分
- 人物生卒年份
- 事件发生的时间顺序

### 3.2 时间线展示格式

时间线使用表格或内联SVG时间轴：

```
| 时间 | 事件 | 代表人物 | 意义 |
|------|------|---------|------|
| 19xx | ... | ... | ... |
```

## 4. 论述题/案例分析处理

### 4.1 识别论述题

- "试述...""论述...""分析...""评价..."等引导词
- 课件中列出的思考题/讨论题
- 案例分析段落

### 4.2 论述题解析模板

```
✏️ 论述题：[题目原文]
答题要点：
  1. [要点1]：[论据支持]
  2. [要点2]：[论据支持]
  ...
答题框架：[建议的论述结构]
常见误区：[学生容易遗漏或搞错的地方]
```

## 5. 文科专用记忆辅助

### 5.1 记忆口诀/顺口溜

对需要记忆的知识序列（如历史分期、分类体系等），生成记忆口诀。要求：
- 押韵顺口
- 每句对应一个知识点
- 标注口诀中每个关键字对应的完整概念

### 5.2 分类体系

文科课程常见的分类用层级列表或树形图展示。

## 6. 理科方法与文科方法的并存

一个学科可能出现"部分文、部分理"的混合情况。判断标准：

**判定为"理科思维"处理的场景**：
- 课程包含数学公式、计算过程
- 课程包含可量化的分析框架
- 课程包含代码/算法
- 课程包含实验设计/数据分析方法

**判定为"文科思维"处理的场景**：
- 课程包含概念辨析、思想史脉络
- 课程包含案例分析（非数值型）
- 课程包含论述题/开放性思考题
- 课程包含法规条文/政策解读

**两者可在同一HTML中共存**：不同章节可以分别采用理科或文科规则，但同一概念不应交叉使用两套规则。
